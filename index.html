Idiom App

A simple phaser game/flashcard app to help non-native English speakers learn idioms. Plus, it has confetti!

Created in two days for the "May the Code Be with You" Hackathon at the Frisco Public Library.

How to Play

Select either quiz mode or flashcard mode, then study those idioms!
